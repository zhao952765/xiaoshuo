# 功能说明 — Private Novel Studio Pro

## 1. 一键推导

输入主题/关键词，AI 自动生成完整小说框架。

**使用流程：**
1. **输入主题**：手动输入或选择 6 组预设模板（都市言情/玄幻修仙/悬疑推理/末世求生/校园青春/穿越异世界）。
2. **选择长度**：3千字 / 3万字 / 10万字 / 50万字 / 100万字。
3. **设置角色数**：男角色数量、女角色数量。
4. **选择标签**（可选）：进入智能标签系统预选标签，回自动带入。
5. **开始推导**：SSE 流式输出，实时查看进度条和当前阶段。
6. **查看结果**：10 个卡片展示（标题/简介/主角/配角/世界观/感情线/剧情大纲/章节目录/第一章正文/统计信息）。
7. **保存项目**：一键保存到全局项目，自动跳转到剧情可视化。

**成人模式**：右上角开关切换，系统提示词、主角设定、配角设定、感情线描述自动适配成人情色文学风格。

**重新推导**：点击"重新推导"仅清除项目数据，保留 AI 模型等全局配置。

---

## 2. 长篇规划

长篇小说分卷分章规划工具。

**核心功能：**
- **AI 大纲生成**：选择目标字数档位 + 卷数 → AI 生成完整卷列表，每卷自动创建空章节。
- **快速规划**：手动一行一个章节快速录入。
- **3 栏布局**：左侧卷列表（含进度条）/ 中间章节列表（含状态标记）/ 右侧章节编辑器。
- **章节编辑器**：大文本域 + 实时字数统计 + 保存按钮。
- **章节状态**：草稿 / 已完成 / 已润色，点击切换。
- **卷管理**：添加/编辑/删除卷，删除卷时级联删除下属章节。
- **AI 章节生成**：全新生成或续写当前章节，支持目标字数和续写方向选择。

---

## 3. 自动续写

基于上下文自动生成后续内容。

**续写模式：**
- **下一章**：基于上一章内容和章节概要继续生成下一章。
- **续写当前章**：在选中的章节末尾追加内容。
- **批量续写**：依次续写多个章节，支持暂停/停止控制。

**选项配置：**
- **续写方向**：自然推进 / 高潮爆发 / 温情过渡 / 悬念铺设 / 反转突变 / 揭秘真相，或自定义方向。
- **目标字数**：4 档可选。
- **批量章数**：2 / 3 / 5 / 10 章。
- 流式实时输出 + 字数统计。

---

## 4. 文本润色

AI 味检测与多风格润色。

**功能说明：**
- **AI 味检测**：基于 `AI_FLAVOR_WORDS` 词库（60+ 典型 AI 词汇），评分 0-100，高亮标记检测到的词汇，分 3 级提示（严重/有/轻微）。
- **润色模式**：普通润色 / 去 AI 味。
- **润色深度**：轻度（修正语法）/ 中度（调整节奏）/ 深度（重构表达）。
- **文风选择**：网文爽文 / 都市现实 / 出版文学 / 悬疑电影 / 日系轻小说 / 细腻情感 / 自定义。
- **来源**：手动粘贴文本，或从已保存的章节中选择。
- **左右对比**：原文 vs 润色结果，直观对比。
- **操作**：应用到章节 / 复制结果 / 导出 Markdown。
- **历史记录**：保存最近润色记录，方便回溯。

---

## 5. 角色管理

全方位角色设定管理。

**功能说明：**
- **统计卡片**：总角色数 / 主角数 / 配角数 / 龙套数。
- **预设模板**：6 组预设角色模板，快速创建常见角色类型。
- **创建/编辑**：
  - 头像选择器：40 个头像选项。
  - 角色类型：主角 / 配角 / 龙套 / 反派。
  - 性格标签：3 类 48 个（正面 16 / 负面 16 / 中性 16），彩色标签展示。
  - 基础信息：姓名/年龄/性别/职业。
  - 详细设定：外貌/性格/背景/能力/目标/声音/内心世界。
- **AI 辅助**：角色生成 / 角色扩展 / 语言风格，流式输出后应用到表单。
- **关系生成**：选择两个角色 + 输入背景 → AI 生成关系描述 → 双向应用到角色关系列表。
- **导出**：单个角色导出 Markdown / 全部角色导出 JSON。

---

## 6. 世界观管理

详细的世界设定管理。

**功能说明：**
- **统计卡片**：总世界观数 / 有规则数 / 有地理数 / 有历史数。
- **预设模板**：6 组预设世界观模板（奇幻帝国/科幻星际/玄幻大陆等）。
- **世界类型**：8 种（奇幻/科幻/玄幻/都市/历史/末世/武侠/自定义）。
- **创建/编辑**：
  - 概述、规则（名称/描述/范围/限制/副作用）、地点（名称/类型/描述/氛围/场景）、时间线（时代/标题/描述/影响）、社会结构、文化风俗、经济体系。
- **AI 扩展**：扩展区域 / 扩展规则 / 扩展历史，输入条件后流式生成，支持应用到表单或直接保存。
- **AI 一键生成**：输入主题和类型，AI 生成完整世界观。
- **导出**：单个导出 Markdown / 全部导出 JSON / 导入 JSON。

---

## 7. 剧情可视化

多维度展示小说结构。

**标签页：**
- **故事梗概**：编辑标题和简介。
- **角色档案**：3 列网格展示所有角色，点击跳转详情。
- **世界观**：完整展示规则/地点/时间线/社会/文化/经济。
- **感情线**：时间轴形式，每个事件带颜色标记（粉=感情/紫=冲突/红=高潮/深红=肉欲），点击编辑类型/标题/描述/关联角色。
- **剧情大纲**：可编辑节点，支持添加/删除/上下排序。
- **章节目录**：列表展示，含字数和状态标记。
- **关系图谱**：React Flow 交互图，节点按角色类型着色（主角紫/反派红/配角粉），边显示关系类型。

---

## 8. 智能标签

标签分类与管理系统。

**功能说明：**
- **统计卡片**：总标签数 / 已选数 / 收藏数 / 分类数。
- **预设标签组**：8 组热门标签组合，一键导入。
- **模式切换**：在线扩展（调用 AI）/ 离线扩展（本地规则库）。
- **关键词输入**：输入关键词 → 智能识别分类 → 自动扩展关联标签。
- **标签操作**：点击选中/收藏/删除，拖拽排序。
- **分类浏览**：7 大分类折叠展开（角色/职业/场景/情节/fetish/服装/幻想），带图标和颜色。
- **已选标签区**：复制/清空/去创作（联动一键推导）。
- **收藏标签区**：常用标签快速访问。
- **导入导出**：JSON 导入导出，自动去重。

## 9. AI 模型配置

管理 AI 服务商和参数。

**功能说明：**
- **内置预设**：DeepSeek / 月之暗面 / 通义千问 / 百度千帆 / 硅基流动 / OpenAI / Anthropic / Google / Azure / Ollama。
- **配置字段**：名称 / Base URL / API Key / 模型类型 / 模型 ID / Temperature / Max Tokens / 流式输出开关。
- **自动名称**：切换模型类型时，如名称为空自动填入当前类型名。
- **测试连接**：验证 API 可用性。
- **获取模型列表**：自动拉取服务商可用模型。
- **持久化**：配置自动保存，启动时自动还原当前模型。

## 10. 设置中心

综合设置与导出。

**Tab 页：**
- **外观设置**：深色/浅色主题切换，3 档字体大小（小/中/大），8 种强调色选择。
- **数据管理**：
  - 5 档自动保存间隔（1/3/5/10/30 分钟）。
  - 9 项数据概览（角色/章节/世界观等数量）。
  - JSON 导入/导出完整项目。
  - 11 项存储详情（含大小计算）。
  - 逐项清除 / 清除全部（带确认弹窗）。
- **生成设置**：Temperature 滑块（0-2），5 档 Max Tokens，流式输出开关，已配置模型列表。
- **隐私安全**：API 密钥显示/隐藏切换，3 条安全提示。
- **关于**：版本信息，12 个功能模块状态，5 条使用提示。

**导出中心（设置中心内）：**
- **统计卡片**：总项目/总章节/总字数/角色数/世界观数。
- **项目选择**：选择要导出的项目。
- **导出格式**：TXT / Markdown / JSON / PDF（浏览器打印）。
- **导出范围**：完整项目 / 按卷导出 / 按章导出 / 角色资料 / 世界观设定。
- **卷选择**：全选/取消。
- **章节选择**：含状态标记和字数显示。
- **导出预览**：弹窗预览内容，支持下载和复制。
- **快捷导出**：角色卡片集 / 世界观设定集 / 完整工程备份 / 纯文本小说。

---

## 11. 记忆系统

自动/手动记录创作过程。

**功能说明：**
- **统计卡片**：记忆总数 / LLM 调用次数 / 已配置模型数 / LLM 总耗时。
- **使用概览**：当前状态 / 资源统计 / LLM 统计 3 列展示。
- **自动记忆**：自动追踪当前模型变化、当前项目变化、最近项目列表、常用标签、LLM 调用统计。
- **手动添加**：输入内容/来源/标签，选择类型后保存。
- **搜索与过滤**：关键词搜索 + 8 种类型过滤（auto/manual/llm/error 等）。
- **记忆列表**：按类型分组，支持折叠/展开/编辑/删除/清空确认。
- **导入导出**：JSON 导入导出（含模型/项目/标签数据）。
- **快捷跳转**：从记忆卡片跳转到相关页面。

---

## 12. AI 对话

携带项目上下文的 AI 助手。

**功能说明：**
- **上下文注入**：自动携带当前项目的角色设定、世界观、章节内容作为对话上下文。
- **多轮对话**：支持连续问答，保持上下文连贯。
- **新建会话**：一键清空重新开始。
- **导出对话**：导出为 Markdown 或纯文本。
