# 功能说明 — Private Novel Studio Pro

## 1. 仪表盘 (`/`)

项目概览与统计数据。

- **项目信息卡**：当前项目标题、简介、模式（常规/成人）、目标字数
- **统计卡片**：总字数、章节数（完成/总数）、角色数、世界观数、标签数、感情节点数、肉欲强度点数、平均强度（成人模式）
- **最近日志**：最近 20 条操作日志，按类型着色（成功绿/错误红/警告橙/信息蓝），可导出 JSON

---

## 2. 一键推导 (`/deduce`)

输入剧情设定，AI 自动推导大纲与详细展开。

**使用流程：**
1. **输入设定**：在文本域中输入故事核心设定、主题、角色关系、世界观等
2. **开始推导**：点击「🎯 开始推导」，AI 流式输出
3. **查看结果**：三段式面板展示——结果头部（字数统计）+ 故事大纲 + 详细展开
4. **重新推导**：修改输入后再次点击

**UI 说明**：
- 深色主题卡片式布局，主色 `#FF4D94`
- 输入区带字数统计
- 结果区带渐入动画

---

## 3. 长篇规划 (`/longplan`)

长篇小说分卷分章规划工具。

**核心功能：**
- **AI 大纲生成**：选择目标字数档位 + 卷数 → AI 生成完整卷列表，每卷自动创建空章节
- **3 栏布局**：左侧卷列表（含进度条）/ 中间章节列表（含状态标记）/ 右侧章节编辑器
- **章节编辑器**：大文本域 + 实时字数统计 + 保存按钮
- **章节状态**：草稿 / 已完成 / 已润色，点击切换
- **卷管理**：添加/编辑/删除卷，删除卷时级联删除下属章节
- **AI 章节生成**：全新生成或续写当前章节，支持目标字数和续写方向选择

---

## 4. 自动续写 (`/continue`)

基于上下文自动生成后续内容。

**续写模式：**
- **下一章**：基于上一章内容和章节概要继续生成下一章
- **续写当前章**：在选中的章节末尾追加内容
- **批量续写**：依次续写多个章节，支持暂停/停止控制

**选项配置：**
- **续写方向**：自然推进 / 高潮爆发 / 温情过渡 / 悬念铺设 / 反转突变 / 揭秘真相，或自定义方向
- **目标字数**：4 档可选（1K/2K/3K/5K/10K）
- 流式实时输出 + 进度条 + 字数统计
- 支持追加保存和替换原文

---

## 5. 文本润色 (`/polish`)

AI 文本润色，提升文笔质量。

**功能说明：**
- **润色模式**：
  - 标准润色：优化表达，保持原意
  - 文学增强：提升文学性
  - 简化表达：使文本更简洁
  - 情绪增强：增强情感张力
- **智能文本分割**：在段落/句子边界分割，不在句子中间截断
- **来源**：手动粘贴文本
- **输出**：润色结果卡片展示，含字数统计

---

## 6. 写作区 (`/write`)

AI 辅助自由创作。

**功能说明：**
- **输入设定**：输入故事设定、角色对话、场景描述
- **生成**：AI 基于输入生成内容
- **续写**：基于已生成内容继续创作
- **清空**：重置写作区
- 使用共享 UI 组件（Card/Btn/Empty）

---

## 7. AI 对话 (`/chat`)

携带项目上下文的 AI 助手。

**功能说明：**
- **对话气泡 UI**：用户消息右对齐（粉色）/ AI 回复左对齐（灰色）
- **Enter 发送**：按 Enter 发送消息，Shift+Enter 换行
- **自动滚动**：新消息自动滚到底部
- **记忆提取**：自动从 AI 回复中提取角色/世界观/剧情记忆
- **空状态**：友好的起始提示

---

## 8. 角色管理 (`/character`)

全方位角色设定管理（53KB 大页面）。

**功能说明：**
- **统计卡片**：总角色数 / 主角数 / 配角数 / 龙套数
- **预设模板**：6 组预设角色模板
- **创建/编辑**：
  - 头像选择器：40 个头像选项
  - 角色类型：主角 / 配角 / 龙套 / 反派
  - 性格标签：3 类 48 个（正面 16 / 负面 16 / 中性 16），彩色标签展示
  - 基础信息：姓名/年龄/性别/职业
  - 详细设定：外貌/性格/背景/能力/目标/声音/内心世界
- **AI 辅助**：角色生成 / 角色扩展 / 语言风格
- **关系生成**：选择两个角色 + 输入背景 → AI 生成关系描述 → 双向应用到角色关系列表
- **导出**：单个角色导出 Markdown / 全部角色导出 JSON

## 9. 世界观管理 (`/world`)

世界观设定管理（重构版）。

**功能说明：**
- **搜索与过滤**：关键词搜索 + 8 种世界类型过滤
- **世界类型**：8 种（校园/都市/末世/奇幻/科幻/玄幻/历史/武侠/自定义）
- **创建/编辑**：
  - 名称、类型选择、简介、详细世界观概述
- **导出**：全部导出 JSON
- **删除**：确认弹窗后删除
- **UI**：使用共享组件（Card/Btn/Modal/Badge），卡片网格布局

## 11. 感情线编辑器 (`emotionFlow`)

React Flow 节点编辑器。

- **4 种节点类型**：感情（粉）/ 冲突（橙）/ 高潮（红）/ 肉欲（紫）
- **拖拽操作**：调整节点位置、拖拽连线连接节点
- **节点编辑**：点击节点编辑名称/类型/强度/描述
- **添加/删除**：工具栏按钮操作
- **保存连线**：显式保存连线到 Store
- **实时同步**：Store 变化自动同步到 Flow 画布

## 12. 肉欲线编辑器 (`lustCurve`)

- **柱状图**：每个章节一个柱子，高度代表强度值
- **拖拽调整**：拖拽圆点调整强度（0-100）
- **高潮标记**：悬浮显示"标记高潮"/"取消高潮"按钮
- **图例**：普通强度 / 高强度(>75) / 高潮点

## 22. 数据持久化机制

**存储方式**：Zustand `persist` 中间件 + `localStorage`

**持久化键名**：`private-novel-studio-pro-storage`

**版本控制**：`_version: 2`
- v0→v1：从旧键 `private-novel-studio-pro-storage` 恢复
- v1→v2：新增 `emotionEvents`/`volumes` 字段

**自动保存**：
- `autoSaveInterval` 定时器（秒级），自动全量写入
- `autoBackup` 启用时保存到 `backup-{timestamp}`，保留 10 个

**双保险保存**：
- `beforeunload` 事件（浏览器刷新/关闭）
- `visibilitychange` 事件（标签页切换）

**数据完整性校验**（`validateDataIntegrity`）：
- 确保所有数组存在
- 修复 EmotionArc/LustArc 空数组
- 修复 deduceTask 类型
- 启动时自动重置 `isRunning=true`（上次推导中断）

**备份工具**：
- `saveNow()` — 手动触发保存
- `listBackups()` — 列出可用备份
- `restoreFromBackup(key)` — 从备份恢复
- `importData(data)` — 导入数据
- `exportData()` — 导出数据（排除函数）
