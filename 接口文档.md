## 5. 数据流

```
用户操作
  → 页面组件调用 Store Action
    → Zustand set() 更新状态
      → persist 中间件自动同步到 localStorage
        → beforeunload / Electron close 事件强制全量写入
```

**持久化键名**: `private-novel-studio-pro-storage`

**双保险机制**:
1. `persist` 中间件：每次 `set()` 后自动写入 localStorage
2. `beforeunload` 事件：页面刷新/关闭前强制写入
3. Electron `close` 事件：窗口关闭时通过 `executeJavaScript` 强制写入

**数据迁移**: `loadProject()` 兼容旧版本字段，自动补充缺失字段默认值。

## 6. AI 调用接口

所有 AI 调用通过标准 HTTP POST + SSE (Server-Sent Events) 实现流式输出。

**请求格式**:
```typescript
{
  model: string;
  messages: Array<{ role: 'system' | 'user' | 'assistant'; content: string }>;
  temperature?: number;
  max_tokens?: number;
  stream?: boolean;
}
```

**响应处理**:
- `stream: true`：逐 token 解析 SSE 数据流，实时更新 UI
- `stream: false`：一次性返回完整文本

**超时控制**: 由 `apiTimeout` 设置控制（默认 60 秒）。
