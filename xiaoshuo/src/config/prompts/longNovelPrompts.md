# 长篇小说管理 - 提示词

## 长篇小说大纲生成提示词

请为以下长篇小说生成详细的卷级大纲：

【小说标题】{title}
【故事主题】{theme}
【目标字数】{targetWords}字
【卷数】{volumeCount}卷

请为每一卷生成：
1. 卷名（有气势、有暗示）
2. 卷概述（100-200字，概括本卷核心事件）
3. 章节规划（每卷{chaptersPerVolume}章，每章含标题+一句话概要）
4. 本卷核心冲突
5. 本卷情感走向
6. 本卷结尾钩子

注意：
- 各卷之间要有递进和转折
- 节奏张弛有度，高潮与过渡交替
- 每卷结尾都要有钩子驱动阅读下一卷
- 人物成长线要贯穿始终

## 单卷详细大纲提示词

请为以下小说的某一卷生成详细的章节大纲：

【小说标题】{title}
【故事主题】{theme}
【卷序号】第{volumeIndex}卷
【卷名称】{volumeName}
【卷概述】{volumeSummary}
【前卷摘要】{previousSummary}

请为每一章生成：
1. 章节标题（引人入胜）
2. 章节概要（50-100字）
3. 出场人物
4. 核心事件
5. 情绪基调（紧张/温馨/悲伤/热血/悬疑等）
6. 章尾钩子

## 章节生成提示词

请根据以下信息撰写一章完整的小说内容：

【小说标题】{title}
【当前卷】{volumeName}
【章节标题】{chapterTitle}
【章节概要】{chapterSummary}
【出场人物】{characters}
【情绪基调】{mood}

【前文回顾】
{previousContext}

【角色设定参考】
{characterRef}

【世界观参考】
{worldviewRef}

撰写要求：
1. 字数：{wordCount}字以上
2. 开篇即抓人，前3句话设置悬念或冲突
3. 人物对话有个人特色，符合角色性格
4. 场景描写服务于氛围和情感
5. 章节末尾设置钩子
6. 保持与前文的连贯性和一致性

## 续写章节提示词（用于自动续写）

请续写当前章节的后续内容。

【小说标题】{title}
【当前章节】{chapterTitle}
【章节概要】{chapterSummary}

【已写内容】
{writtenContent}

【待续写方向】
{direction}

续写要求：
1. 与已写内容自然衔接
2. 保持人物性格和文风一致
3. 推进剧情发展
4. 续写约{wordCount}字
